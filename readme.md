# Animated Login & Signup UI

A modern and responsive Login and Signup Authentication Interface built using HTML, Tailwind CSS, and JavaScript. The project provides a clean and minimal user interface with smooth animations for switching between the Sign In and Sign Up forms.

## Description

This project demonstrates how to create an animated authentication page using HTML, Tailwind CSS, and JavaScript. The interface uses CSS transitions and JavaScript class toggling to switch between login and registration forms without reloading the page.

It is suitable for beginners who want to learn responsive UI design and form animations, and it can also be used as a starting template for web applications.

## Features

- Modern and clean user interface
- Responsive design for desktop and mobile devices
- Smooth animations between Sign In and Sign Up forms

## How It Works

- The Sign In form is displayed when the page loads.
- Clicking the Sign Up button displays the registration form.
- Clicking the Sign In button returns to the login form.
- JavaScript adds and removes CSS classes to animate the transition.
- CSS handles the panel movement and transition effects.

## How to Run

1. Download or clone the project.
2. Make sure the main file is named `index.html`.
3. Open `index.html` in any modern web browser.

No installation or additional software is required.

## Deployment on GitHub Pages

1. Create a new GitHub repository.
2. Upload the project files.
3. Open the repository Settings.
4. Select Pages.
5. Under Source, choose Deploy from a branch.
6. Select the `main` branch and the root folder.
7. Save the changes.

Your website will be available at:

https://USERNAME.github.io/REPOSITORY-NAME/
