````md
# 🔐 Animated Login & Signup UI

This project is a modern and responsive **Login & Signup authentication interface** built using **HTML, Tailwind CSS, and JavaScript**. It features smooth sliding animations, clean UI transitions, form switching effects, and a minimal authentication layout for modern web applications.

The UI includes two forms:
- **Sign In**
- **Sign Up**

and dynamically switches between them using CSS animations and JavaScript class toggling. :contentReference[oaicite:0]{index=0}

---

## ✨ Features

✔ Modern glass-like UI  
✔ Tailwind CSS styling via CDN  
✔ Smooth animated transitions  
✔ Form switch animation using JavaScript  
✔ Fully responsive layout  
✔ Clean and minimal authentication design

Animations and transitions are controlled by CSS classes such as:
- `right-panel-active`
- `overlay-panel`
- `overlay-left`
- `overlay-right` :contentReference[oaicite:1]{index=1}

The switching is triggered by two buttons inside JS:
```js
signUpButton.addEventListener('click', () => {
    container.classList.add('right-panel-active');
});
signInButton.addEventListener('click', () => {
    container.classList.remove('right-panel-active');
});
``` :contentReference[oaicite:2]{index=2}
````

```
## ▶ How It Works

- By default, the **Sign In** form is visible  
- Clicking **Sign Up** moves UI panel using CSS transforms
- CSS animations are handled by:
  - `.right-panel-active`
  - overlay panels
  - keyframe animation: `@keyframes show`  
- Tailwind handles typography, spacing, colors, borders and responsive layout
```
